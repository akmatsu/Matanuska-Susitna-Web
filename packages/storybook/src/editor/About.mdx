# MSB UI Editor

The MSB UI Editor is the main drag and drop block style UI Editor for the Matanuska Susitna Borough. Any pages that allow drag and drop editing are using this under the hood.

## Setup

### Installation

Be sure to install required peer dependencies

```bash
pnpm i -F <your-project-name> react next
```

Then install the package:

```bash
pnpm i -F <your-project-name> @matsugov/ui-editor
```

### Styles

This packages uses TailwindCSS so be sure to setup your project's `tailwind.config.js` to pick up CSS classes otherwise things may not render correctly:

```js
// tailwind.config.js
const path = require('path');

/** @type {import('tailwindcss').Config} */
module.exports = {
  content: [
    './src/**.*.{js,ts,jsx,tsx}',
    path.join(
      path.dirname(require.resolve('@matsugov/ui-editor')),
      '**/*.{js,ts,jsx,tsx}',
    ),
  ],
  presets: [require('@matsugov/configs/tailwind.config')],
};
```

## Usage

To use this component be sure you followed all of the [setup instructions](#setup) above and then you can just import it into your react page/component

```tsx
import { CoreEditor } from '@matsugov/ui-editor';

export default function Home() {
  return (
    <main>
      <CoreEditor />
    </main>
  );
}
```
